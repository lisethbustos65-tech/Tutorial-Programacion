¿Qué es una lista?

Una lista es una estructura de datos que permite almacenar varios elementos. En Java se utiliza la clase ArrayList para crear listas dinámicas.

A diferencia de los arreglos tradicionales, una lista puede aumentar o disminuir de tamaño automáticamente.

¿Para qué sirven las listas?

  • Guardar información de forma dinámica.
  • Agregar y eliminar elementos fácilmente.
  • Almacenar colecciones de datos.
  • Gestionar registros de usuarios, productos o estudiantes.

Crear una lista

import java.util.ArrayList;

public class Lista {

    public static void main(String[] args) {

        ArrayList nombres = new ArrayList<>();

        nombres.add("Ana");
        nombres.add("Luis");
        nombres.add("Carlos");

        System.out.println(nombres);

    }

}

Explicación

  • ArrayList crea una lista dinámica.
  • String indica que almacenará texto.
  • add() agrega elementos.
  • System.out.println() muestra la lista.

Acceder a un elemento

System.out.println(nombres.get(0));

El método get() permite acceder a un elemento mediante su posición.

Eliminar un elemento

nombres.remove(1);

Elimina el elemento que se encuentra en la posición indicada.

Ejercicio Propuesto

Crear una lista de cinco ciudades y mostrar todas en pantalla.

Solución

ArrayList ciudades = new ArrayList<>();

ciudades.add("Quito");
ciudades.add("Guayaquil");
ciudades.add("Cuenca");
ciudades.add("Manta");
ciudades.add("Loja");

System.out.println(ciudades);

Errores Comunes

  • Olvidar importar ArrayList.
  • Intentar acceder a una posición que no existe.
  • Confundir arreglos con listas.
