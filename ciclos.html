Ciclos en Java

Los ciclos permiten repetir instrucciones varias veces sin escribir el mismo código repetidamente.

Ciclo for

El ciclo for se utiliza cuando conocemos cuántas veces queremos repetir una acción. 

public class CicloFor {

    public static void main(String[] args) {

        for(int i = 1; i <= 5; i++) {
            System.out.println(i);
        }

    }
}

Explicación

  • int i = 1: inicia la variable en 1.
  • i <= 5: el ciclo se ejecuta mientras i sea menor o igual a 5.
  • i++: aumenta el valor de i en 1 después de cada repetición.

Salida: 

1
2
3
4
5
