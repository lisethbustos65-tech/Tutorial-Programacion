📌 ¿Qué es el manejo de archivos?

Es la capacidad de un programa para almacenar información de forma permanente en el computador, permitiendo que los datos no se pierdan al cerrar el programa.

🎯 ¿Para qué sirve?

  • Guardar usuarios o registros.
  • Almacenar inventarios o ventas.
  • Guardar configuraciones del sistema.
  • Leer información previamente guardada.

✍️ Escribir en un archivo

import java.io.FileWriter;
import java.io.IOException;

public class EscribirArchivo {

    public static void main(String[] args) throws IOException {

        FileWriter archivo = new FileWriter("datos.txt");

        archivo.write("Hola Mundo");

        archivo.close();
    }
}

Explicación:

  • FileWriter: crea o abre un archivo.
  • write(): escribe datos dentro del archivo.
  • close(): guarda y cierra correctamente.

📖 Leer un archivo

import java.io.File;
import java.util.Scanner;

public class LeerArchivo {

    public static void main(String[] args) throws Exception {

        File archivo = new File("datos.txt");

        Scanner lector = new Scanner(archivo);

        while(lector.hasNextLine()) {
            System.out.println(lector.nextLine());
        }

        lector.close();
    }
}

🧠 Ejercicio Propuesto

Crea un archivo llamado estudiantes.txt y guarda tres nombres.

✅ Solución

FileWriter archivo = new FileWriter("estudiantes.txt");

archivo.write("Ana\n");
archivo.write("Carlos\n");
archivo.write("María\n");

archivo.close();

⚠️ Errores comunes

  • No cerrar el archivo.
  • No importar librerías.
  • Ruta incorrecta del archivo.
  • No manejar excepciones.
