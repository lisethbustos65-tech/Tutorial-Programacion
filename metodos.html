Métodos en Java

Un método es un bloque de código que realiza una tarea específica. Su principal ventaja es evitar repetir código.

Ejemplo de método

public class Metodos {

    public static void saludar() {
        System.out.println("Hola, bienvenido a Java");
    }

    public static void main(String[] args) {

        saludar();

    }
}

Explicación

  • public: permite acceder al método desde cualquier lugar.
  • static: permite usar el método sin crear un objeto.
  • void: indica que no devuelve ningún valor.
  • saludar(): nombre del método.
  • saludar(); ejecuta el método.

Salida

Hola, bienvenido a Java

Método con retorno

public static int sumar(int a, int b){
    return a + b;
}

La palabra return devuelve un resultado.
