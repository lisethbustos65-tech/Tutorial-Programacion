Manejo de Archivos en Java

El manejo de archivos permite guardar información de forma permanente en el computador.

Escribir un archivo

import java.io.FileWriter;
import java.io.IOException;

public class EscribirArchivo {

    public static void main(String[] args) throws IOException {

        FileWriter archivo = new FileWriter("datos.txt");

        archivo.write("Hola Mundo");

        archivo.close();

    }
}

Explicación

  • FileWriter permite escribir en un archivo.
  • write() escribe el contenido.
  • close() guarda y cierra el archivo.

Leer un archivo

import java.io.File;
import java.util.Scanner;

public class LeerArchivo {

    public static void main(String[] args) throws Exception {

        File archivo = new File("datos.txt");

        Scanner lector = new Scanner(archivo);

        while(lector.hasNextLine()) {
            System.out.println(lector.nextLine());
        }

        lector.close();

    }
}

Conceptos importantes

  • Scanner permite leer información.
  • hasNextLine() verifica si quedan líneas por leer.
  • nextLine() obtiene la siguiente línea.
