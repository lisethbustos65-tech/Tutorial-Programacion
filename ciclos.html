¿Qué es un ciclo?

Un ciclo es una estructura de control que permite ejecutar varias veces un mismo bloque de instrucciones sin necesidad de repetir el código manualmente.

Los ciclos son fundamentales en programación porque permiten automatizar tareas repetitivas y procesar grandes cantidades de información de manera eficiente.

¿Para qué sirven los ciclos?

  • Mostrar secuencias numéricas.
  • Recorrer vectores y matrices.
  • Procesar listas de datos.
  • Automatizar tareas repetitivas.
  • Reducir la cantidad de código.

Tipos de ciclos en Java

  • for
  • while
  • do-while

Ciclo For

El ciclo for se utiliza cuando conocemos de antemano cuántas veces queremos repetir una acción. 

public class CicloFor {

    public static void main(String[] args) {

        for(int i = 1; i <= 5; i++) {

            System.out.println(i);

        }

    }

}

Explicación del código

int i = 1

Crea una variable llamada i que inicia con el valor 1.

i <= 5

Mientras esta condición sea verdadera el ciclo seguirá ejecutándose.

i++

Incrementa el valor de i en una unidad después de cada repetición.

System.out.println(i)

Muestra el valor actual de la variable i.

Resultado

1
2
3
4
5

Ciclo While

El ciclo while se utiliza cuando no sabemos exactamente cuántas veces se repetirá una acción. 

public class CicloWhile {

    public static void main(String[] args) {

        int numero = 1;

        while(numero <= 5){

            System.out.println(numero);

            numero++;

        }

    }

}

El ciclo continuará ejecutándose mientras la condición sea verdadera.

Ciclo Do While

La diferencia principal es que el bloque de instrucciones se ejecuta al menos una vez antes de verificar la condición. 

public class CicloDoWhile {

    public static void main(String[] args) {

        int numero = 1;

        do{

            System.out.println(numero);

            numero++;

        }while(numero <= 5);

    }

}

Ejercicio Propuesto

Realiza un programa que muestre los números del 1 al 10 utilizando un ciclo for.

Solución

public class Ejercicio1 {

    public static void main(String[] args) {

        for(int i = 1; i <= 10; i++) {

            System.out.println(i);

        }

    }

}

Errores comunes de los principiantes

  • Olvidar incrementar la variable del ciclo.
  • Escribir incorrectamente la condición.
  • Crear ciclos infinitos.
  • Confundir el operador = con ==.
