Listas (ArrayList) en Java

Un ArrayList permite almacenar elementos dinámicamente. A diferencia de los arreglos, puede crecer o disminuir de tamaño.

Ejemplo

import java.util.ArrayList;

public class Lista {

    public static void main(String[] args) {

        ArrayList nombres = new ArrayList<>();

        nombres.add("Ana");
        nombres.add("Luis");
        nombres.add("Carlos");

        System.out.println(nombres);

    }
}

Explicación

  • ArrayList es una colección dinámica.
  • add() agrega elementos.
  • String indica que la lista almacenará texto.
  • No es necesario definir el tamaño al crearla.

Salida

[Ana, Luis, Carlos]
