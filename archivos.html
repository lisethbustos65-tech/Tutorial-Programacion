¿Qué es el manejo de archivos?

El manejo de archivos permite almacenar información de forma permanente en el computador. Gracias a esto, los datos pueden conservarse incluso después de cerrar el programa.

¿Para qué sirve?

  • Guardar información de usuarios.
  • Registrar ventas o inventarios.
  • Almacenar configuraciones.
  • Leer datos previamente guardados.

Escribir en un archivo

import java.io.FileWriter;
import java.io.IOException;

public class EscribirArchivo {

    public static void main(String[] args) throws IOException {

        FileWriter archivo = new FileWriter("datos.txt");

        archivo.write("Hola Mundo");

        archivo.close();

    }

}

Explicación

  • FileWriter crea o abre un archivo.
  • write() escribe información.
  • close() guarda y cierra el archivo.

Leer un archivo

import java.io.File;
import java.util.Scanner;

public class LeerArchivo {

    public static void main(String[] args) throws Exception {

        File archivo = new File("datos.txt");

        Scanner lector = new Scanner(archivo);

        while(lector.hasNextLine()) {

            System.out.println(lector.nextLine());

        }

        lector.close();

    }

}

Explicación

  • File representa el archivo.
  • Scanner permite leer información.
  • hasNextLine() verifica si existen más líneas.
  • nextLine() obtiene la siguiente línea.

Ejercicio Propuesto

Crear un archivo llamado estudiantes.txt y guardar tres nombres de estudiantes.

Solución

FileWriter archivo = new FileWriter("estudiantes.txt");

archivo.write("Ana\n");
archivo.write("Carlos\n");
archivo.write("María\n");

archivo.close();

Errores Comunes

  • Olvidar cerrar el archivo.
  • No importar las librerías necesarias.
  • Escribir mal la ruta del archivo.
  • No manejar excepciones.
