¿Qué es un método?

Un método es un conjunto de instrucciones agrupadas bajo un nombre específico para realizar una tarea determinada.

Los métodos permiten reutilizar código, mejorar la organización del programa y facilitar el mantenimiento de aplicaciones grandes.

¿Para qué sirven los métodos?

  • Evitan repetir código.
  • Facilitan la lectura de programas.
  • Permiten dividir problemas complejos en tareas pequeñas.
  • Mejoran la reutilización del código.
  • Hacen que los programas sean más fáciles de mantener.

Estructura General de un Método

public static void saludar(){

    System.out.println("Hola");

}

Significado de cada elemento

  • public: permite acceder al método desde cualquier parte del programa.
  • static: permite utilizar el método sin crear un objeto.
  • void: indica que el método no devuelve ningún valor.
  • saludar: es el nombre del método.
  • () contiene los parámetros que puede recibir.

Método sin Parámetros

public class MetodoSimple {

    public static void saludar(){

        System.out.println("Bienvenido a Java");

    }

    public static void main(String[] args){

        saludar();

    }

}

En este ejemplo el método saludar() muestra un mensaje en pantalla y es llamado desde el método principal main().

Método con Parámetros

Los parámetros permiten enviar información al método. 

public class MetodoParametros {

    public static void saludar(String nombre){

        System.out.println("Hola " + nombre);

    }

    public static void main(String[] args){

        saludar("Carlos");

    }

}

Cuando se ejecuta el programa, el valor "Carlos" se almacena en el parámetro nombre.

Método con Valor de Retorno

Un método puede devolver información utilizando la palabra reservada return. 

public class MetodoRetorno {

    public static int sumar(int a, int b){

        return a + b;

    }

    public static void main(String[] args){

        int resultado = sumar(5,3);

        System.out.println(resultado);

    }

}

Explicación

  • int indica que el método devolverá un número entero.
  • return devuelve el resultado.
  • El valor retornado se guarda en la variable resultado.

Ejercicio Propuesto

Crear un método llamado multiplicar que reciba dos números y devuelva el resultado de la multiplicación.

Solución

public class EjercicioMetodo {

    public static int multiplicar(int a, int b){

        return a * b;

    }

    public static void main(String[] args){

        System.out.println(multiplicar(4,5));

    }

}

Resultado: 

20

Errores Comunes

  • Olvidar escribir return cuando el método debe devolver un valor.
  • Llamar al método con parámetros incorrectos.
  • Escribir mal el nombre del método.
  • Confundir variables con métodos.
